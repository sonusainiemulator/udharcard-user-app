<p align="center">
  A modern, fast, and secure digital wallet application built with Flutter. <br>
  Designed to bring seamless financial transactions to your fingertips.
</p>

---

## 🚀 Key Features

* **💸 Send & Request Money:** Quickly transfer funds to friends and family, or send a money request with a single tap.
* **🧾 Bill Payments:** Pay utilities, mobile recharges, and other bills effortlessly.
* **📸 QR Code Payments:** Scan and pay instantly at local merchants using built-in QR code scanning capabilities.
* **💳 Vouchers & Redeem Codes:** Apply promotional vouchers and redeem reward codes directly within the app.
* **🏦 Seamless Withdrawals:** Easily withdraw your digital funds to your bank account (integrated with FlutterWave and other payment gateways).
* **🔒 High Security (KYC & 2FA):** Comprehensive Identity Verification and Two-Factor Authentication (2FA) mechanisms to keep your money safe.
* **🎫 Support Tickets:** Built-in customer support ticketing system to resolve queries efficiently.
* **🔔 Smart Notifications:** Real-time push notifications for all your transaction updates and promotional alerts.

## 🛠️ Technology Stack

* **Framework:** Flutter (v3.24+)
* **Language:** Dart 3+
* **State Management:** GetX
* **Local Storage:** Hive / Shared Preferences
* **Payment Gateways:** Stripe, Razorpay, FlutterWave
* **Architecture:** MVC / Modular component-based UI

## ⚙️ Getting Started

### Prerequisites

Ensure you have the following installed on your local machine:
* [Flutter SDK](https://docs.flutter.dev/get-started/install) (Version 3.24 or higher)
* Android Studio / Xcode
* Git

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sonusainiemulator/udharcard-user-app.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd udharcard-user-app
   ```
3. **Install dependencies:**
   ```bash
   flutter pub get
   ```
4. **Run the app:**
   ```bash
   flutter run
   ```

*(Note: Ensure you have your `local.properties` or `.env` files set up with the necessary API keys for Stripe, Razorpay, etc., before running the app).*
